Homophonic Cipher Encoder










Information

This section contains information about how to use this application.

  • Default Key: Default key is used to encode & decode strings if this checkbox is checked. Otherwise a custom key is generated.
  • Phonogram Length: Specifies the length of a single phonogram,
  • Min Phonograms: Minimal phongrams number assigned to a single character.
  • Max Phonograms: Maximal number of phonograms assigned to a single character.
  • Preserve Key: If this field is checked, the key will not be regenerated regardless of other settings.
  • Mode: Specifies operation mode (encoding / decoding).
  • Input: Put your phrase which you want to encode / decode in here.
  • Run: Runs the encoding / decoding operation on string from the input field.
  • View History: Displays the history of operations and some statistical info.
